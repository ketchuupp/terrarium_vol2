<h1>Terrarium control using the STM32 microcontroller</h1>

<h2>Elements used in the project</h2>
<ul>
	<li>STM32F103C8Tx</li>
	<li>LCD 1.8inch ST7735S</li>
	<li>Sensor BME280</li>
	<li>Optotriac moc3021 3x</li>
	<li>Triac BT138 3x</li>
	<li>Source 3.3V</li>
	<li>Buttons 4</li>
</ul>
